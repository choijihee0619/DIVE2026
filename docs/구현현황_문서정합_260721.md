**(a) 개명·대체됨 — 기능은 존재 (6건).** 문서만 낡았다.

| `POST /ml/counsel/predict` | `POST /ml/counsel/classify` |
| `GET /verifications/{verification_id}` | `GET /verifications/{evidence_id}` (경로변수 개명 — evidence 1:1 관계 반영) |
| `POST /verifications/{verification_id}/decision` | `POST /verifications/{evidence_id}/decision` |

**(b) 순수 미구현 (10건).** 260718 재정비 기획에서 기각·보류됐거나 우선순위에서 밀린 것들.
