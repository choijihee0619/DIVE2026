## 사용 원칙 (중요)

1. 이 명단은 임차인 보호 목적으로 **법률에 따라 공개된 정보**다. 서비스 내 용도는
   위험진단의 "임대인 신호" 매칭(성명+주소 일치 여부 확인)으로 한정한다.
2. 포털 원문 주의사항 준수: 명단 정보를 이용해 특정인의 명예를 훼손하면 형사·민사
   책임이 발생할 수 있다. **명단 자체를 UI에 나열·검색 제공하지 말고**, 사용자가 입력한
   임대인 정보와의 일치 여부(true/false + 근거 법령·기준일)만 반환한다.
3. 소명 절차로 삭제될 수 있으므로 `base_date`를 항상 함께 표기하고, 데모 시연 시에는
   실명 대신 마스킹(강○혜)한다.
4. 임베딩·RAG 코퍼스에 포함하지 않는다(개인정보).
